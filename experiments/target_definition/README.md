# Target Definition Experiments

Compare only the two agreed target variants:

1. Primary: next-month entrapment or at least two unscheduled outages.
2. Secondary: next-month entrapment or at least one unscheduled outage.

Questions:

- Which definition represents an operationally meaningful intervention?
- How does prevalence change?
- Are validation performance and calibration stable?
- Does the easier-to-predict target provide less useful prioritization?

Do not select the target using frozen-test results.
